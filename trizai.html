About

What is Trizai?

Trizai is not a single AI agent — it is a composed development team running as a modular, Markdown-driven system on opencode. It reads its intelligence from a hierarchy of instruction modules — rules, core logic, role personas, personality, skills, memory, workflows, and context — each defined in individual Markdown files.

When you ask Trizai to build a UI, it loads the Frontend Developer role. When you need a deployment pipeline, the DevOps Engineer persona activates. Each role is trained to believe it is the world's best in its domain, with deep technical expertise and strict quality standards.

Trizai is an architecture — a reusable system for building AI agents that are transparent, predictable, auditable, and version-controllable. Every file is plain Markdown. Every role has bounded expertise. Every decision is documented.

Getting Started

Launch Trizai in 4 Steps

1

Clone or Copy the Workspace

Download the Trizai folder to your machine. The entire system is portable — just copy the Triz/ directory anywhere.

2

Install OpenCode

Run npm install -g opencode-ai to install the runtime that powers Trizai. Requires Node.js 18+.

3

Configure Your API Key

Run opencode providers set opencode-go <your-api-key> to authenticate. Trizai uses the model configured in opencode.json.

4

Launch Trizai

Run opencode C:\Triz from the command line. The harness in CLAUDE.md will load all modules and you are ready. Trizai automatically detects your task and loads the right expert role.
