# Role Info — Individual Role Pages

This folder contains a dedicated HTML page for every Trizai role. Each page explains that role's **expertise, principles, and mindset** in detail.

## Why `Info/index.html` and not `Info.html`?

The file `Info/index.html` is the **main landing page** for the entire Trizai system — it provides:
- A system overview
- All roles at a glance
- All skills organized by domain
- Architecture diagram
- Local tools reference
- Priority stack showcase
- Quick setup guide

This `Role Info/` folder goes **deeper** — each role gets its own full page dedicated to explaining its capabilities, so you can explore them one at a time.
